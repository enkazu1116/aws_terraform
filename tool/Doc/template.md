# Terraform
## Template構成作成ツール
Zigを使用して、フォルダ構成を自動で作成する。

## 構成
```tree
.
├── env
│   └── dev
│       ├── backend.tf
│       ├── locals.tf
│       ├── main.tf
│       ├── outputs.tf
│       ├── providers.tf
│       ├── terraform.tf
│       ├── terraform.tfvars
│       └── variables.tf
└── module
    ├── main.tf
    ├── outputs.tf
    └── variables.tf
```

## 設計
### 環境ごとのフォルダ作成
envフォルダとその配下に指定フォルダを作成する。
指定フォルダはユーザーに名前を入力させる。
指定フォルダにテンプレートを作成する。

### 子モジュールごとのフォルダ作成
moduleフォルダとその配下に指定フォルダを作成する。
指定フォルダはユーザーに名前を入力させる。
指定フォルダにテンプレートファイルを作成する。

## 詳細設計
処理内容: ルートディレクトリを指定。
1. envフォルダかmoduleフォルダを呼び出すコマンド指定。
引数に応じて処理を分岐させたい。
そのため、コマンドの引数で処理を分岐させる。

2. envフォルダかmoduleフォルダがあるか判定
当該フォルダがある場合は、この処理は不要。
ない場合は、当該フォルダを作成する。

3. 指定フォルダの作成
コマンドで指定された引数を元にフォルダ名を作成する

4. テンプレートファイル構成を作成
指定ごとに事前に定義したファイル構成を作成する。
